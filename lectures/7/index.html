All about Bootstrap

What is Bootstrap?

Bootstrap is a free and open source front-end web development framework for designing webpages faster and easier.

Bootstrap provides a set of HTML- and CSS-based design templates (backed with a little magic of JavaScript) for typography, forms, buttons, tables, navigation, modals, image carousels among many others. Bootstrap also simplifies the creation responsive designs, which automatically adjust themselves to look good on all devices, from small phones to large desktops.

Reasons to Use Bootstrap

Bootstrap helps web developers to easily bootstrap the creation of fluid, simple, yet fully-capable and responsive designs.

In fact, even this page itself is made in Bootstrap! Aside from that, here are some other reasons why Bootstrap really is the one of the best tools to for starter websites:

  • Easy to use. Anybody with just basic knowledge of HTML and CSS can start using Bootstrap.
  • Responsive features. Bootstrap's responsive CSS adjusts to phones, tablets, and desktops.
  • Mobile-first approach. In Bootstrap, mobile-first styles are part of the core framework.
  • Browser compatibility. Bootstrap 5 is compatible with all modern browsers (Chrome, Firefox, Edge, Safari, and Opera). Note that if you need support for IE11 and down, you must use either BS4 or BS3.

Using Bootstrap Containers

Containers are a fundamental building block of Bootstrap that contain, pad, and align your content within a given device or viewport.

Containers are the most basic layout element in Bootstrap and are required when using Bootstrap's default grid system. Choose from a responsive, fixed-width container (meaning its max-width changes at each breakpoint) or fluid-width (meaning it's 100% wide all the time). While containers can be nested, most layouts do not require a nested container.

The simplest way to use Bootstrap containers are by simply using the .container class. Use .container-fluid for a full width container, spanning the entire width of the viewport, as follows: 

<div class="container">
  <!-- Content here -->
</div>
<div class="container-fluid">
  <!-- Content here -->
</div>

Bootstrap comes with three different containers:

  • .container, which sets a max-width at each responsive breakpoint. The default .container class is a responsive, fixed-width container, meaning its max-width changes at each breakpoint.
  • .container-{breakpoint}, which is width: 100% until the specified breakpoint. Responsive containers allow you to specify a class that is 100% wide until the specified breakpoint is reached, after which we apply max-widths for each of the higher breakpoints. For example, .container-sm is 100% wide to start until the sm breakpoint is reached, where it will scale up with md, lg, xl, and xxl.
  • .container-fluid, which is width: 100% at all breakpoints. Use .container-fluid for a full width container, spanning the entire width of the viewport.

Typography with Bootstrap

Bootstrap also provides classes for typography, which can be used to style your text.

Bootstrap provides a handful of classes for styling your text. These classes can be used to set font size, hierarchy, weight, styling, and more. It includes styles for display, headings, paragraphs, leads, lists, blockquotes, etc.

All HTML headings, <h1> through <h6>, are available. .h1 through .h6 classes are also available, for when you want to match the font styling of a heading but cannot use the associated HTML element.

Source

<h1>h1. Bootstrap heading</h1>
<h2>h2. Bootstrap heading</h2>
<h3>h3. Bootstrap heading</h3>
<h4>h4. Bootstrap heading</h4>
<h5>h5. Bootstrap heading</h5>
<h6>h6. Bootstrap heading</h6>

You can also make a leading paragraph stand out using the lead class.

Source

<p class="lead">
  This is a lead paragraph. It stands out from regular paragraphs.
</p>

Output

This is a lead paragraph. It stands out from regular paragraphs.

Colors with Bootstrap

Bootstrap provides a complementary set of colors for use in consistent, semantic designs.

Bootstrap's color classes can be used to style links, buttons, and more. These colors are categorized according to purpose, such as: primary, secondary, success, warning, danger, info, muted, etc. The following examples show how to use them to provide semantic styling.

Source

<p class="text-primary">.text-primary</p>
<p class="text-secondary">.text-secondary</p>
<p class="text-success">.text-success</p>
<p class="text-danger">.text-danger</p>
<p class="text-warning">.text-warning</p>
<p class="text-info">.text-info</p>

<p class="text-dark">.text-dark</p>
