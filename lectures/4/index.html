Anchors

The HTML anchor (<a>) tag defines a hyperlink. It has the following syntax:

<a href="url">link text</a>

The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

The link text is the part that will be visible to the reader. Clicking on the link text, will send the reader to the specified URL address.

Anchor Attributes

The <a> tag has one required attribute:

href
Stands for hyperreference, this specifies the path where the anchor links to.

Note: When a web page loads, it is the browser, at that moment, that gets the image from a web server and inserts it into the page. Therefore, make sure that the image actually stays in the same spot in relation to the web page, otherwise your visitors will get a broken link icon. The broken link icon and the alt text are shown if the browser cannot find the image.

Other attributes for the <a> tag include:

target
This specifies where to open the linked document. It can take the following values:
_self
Default. Opens the document in the same window/tab as it was clicked
_blank
Opens the document in a new window or tab
_parent
Opens the document in the parent frame
_top
Opens the document in the full body of the window
