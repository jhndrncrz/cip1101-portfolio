Images

The HTML image (<img>) tag is used to embed an image in a web page. It has the following syntax:

<img src="url" alt="text here">

Images are not technically inserted into a web page; images are linked to web pages. The <img> tag creates a holding space for the referenced image.

The <img> tag is empty; it contains attributes only and does not have a closing tag, though it can be self-closed (following XHTML standards).

Image Attributes

The <img> tag has two required attributes:

src
Stands for source, this specifies the path to the image.

Note: When a web page loads, it is the browser, at that moment, that gets the image from a web server and inserts it into the page. Therefore, make sure that the image actually stays in the same spot in relation to the web page, otherwise your visitors will get a broken link icon. The broken link icon and the alt text are shown if the browser cannot find the image.

alt
Stands for alternate, this specifies an alternate text for the image.

Note: If the user for some reason cannot view an image (i.e., because of slow connection, an error in the src attribute, or if the user uses a screen reader), the alt attribute takes its place instead. Thus, the value of the alt attribute should describe the image.

Images with Anchors

You can make an image act as a hyperlink by simply enclosing it in an <a> tag.
